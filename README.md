# GianGen

A dungeon generator written in Python.

Uses [Pillow](https://pillow.readthedocs.io/en/3.0.x/handbook/tutorial.html) image manipulation library.


## Instructions

### Prerequisites
- Python 3
- pipenv

### Installation
`> git clone https://github.com/angian00/giangen`

`> cd giangen`

`> pipenv install '-e ./src/'`

### Execution
`> ./run.sh`
